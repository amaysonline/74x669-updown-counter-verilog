# 74x669 Synchronous Up/Down Counter (Verilog)

A Verilog implementation of the **74x669 synchronous 4-bit up/down binary counter**, designed, simulated, and implemented on a **Spartan-7 Boolean Board** using **Xilinx Vivado 2020.2**.

---

## Project Overview

This project recreates the functionality of the **74x669 TTL Integrated Circuit** using Verilog HDL. The design supports synchronous counting, synchronous clear, synchronous parallel loading, and ripple carry generation.

The project was:

- Designed in Verilog HDL
- Verified using a comprehensive testbench
- Simulated in Xilinx Vivado
- Synthesized and implemented on a Spartan-7 FPGA

---

## Features

- 4-bit synchronous binary counter
- Up and Down counting
- Active-low synchronous clear
- Active-low synchronous parallel load
- Dual active-low enable inputs
- Active-low Ripple Carry Output (RCO)
- FPGA implementation on Spartan-7
- Fully synthesizable Verilog
- Self-checking testbench

## Inputs

| Signal | Description |
|---------|-------------|
| clk | System Clock |
| clr_n | Active-Low Synchronous Clear |
| load_n | Active-Low Parallel Load |
| ent_n | Enable T (Active-Low) |
| enp_n | Enable P (Active-Low) |
| updown | Count Direction (1 = Up, 0 = Down) |
| d[3:0] | Parallel Data Input |

---

## Outputs

| Signal | Description |
|---------|-------------|
| q[3:0] | Counter Output |
| rco_n | Active-Low Ripple Carry Output |

---

## Functional Priority

The counter operates according to the following priority:

1. Synchronous Clear
2. Parallel Load
3. Count Up / Down
4. Hold State

## FPGA Implementation

**Target FPGA Board**

- Boolean Board
- Spartan-7 (XC7S50CSGA324-2)

**Toolchain**

- Xilinx Vivado 2020.2

### Button Mapping

| Button | Function |
|---------|----------|
| BTN0 | Synchronous Clear |
| BTN1 | Parallel Load |
| BTN2 | Count Direction (Up/Down) |
| BTN3 | Unused |

### Switch Mapping

| Switch | Function |
|---------|----------|
| SW[3:0] | Parallel Load Data |

### LEDs

| LEDs | Description |
|------|-------------|
| LED[3:0] | Counter Output |

## Running the Project

1. Open the Vivado project.
2. Add the source files.
3. Add the constraints file.
4. Run Behavioral Simulation.
5. Run Synthesis.
6. Generate Bitstream.
7. Program the Spartan-7 FPGA.
